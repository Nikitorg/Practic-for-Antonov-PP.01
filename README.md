# MFCFuture

## Описание
MFCFuture — это приложение для онлайн-записи на прием в МФЦ.  

## Функциональные возможности
- Просмотр списка доступных МФЦ
- Выбор государственной услуги
- Онлайн-запись на прием
- Выбор даты и свободного времени
- Хранение истории записей пользователя

## Используемые технологии
- Язык программирования: Java
- База данных: -
- Интерфейс: Крутой
